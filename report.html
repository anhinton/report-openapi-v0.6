OpenAPI version 0.6

Ashley Noel Hinton and Paul Murrell
date

The OpenAPI architecture specifies a method for sharing and combining pieces of data analysis code so as to help people connect with data. OpenAPI describes how pieces of data analysis code can be wrapped in modules, and how these modules can be combined in pipelines which describe a whole data analysis workflow. Modules and pipelines are created using simple XML, and are intended to promote sharing and re-use. Modules and pipelines can be executed in OpenAPI glue system software (.

This report describes the changes to the OpenAPI architecture in version 0.6.
